# RBAC System API (Flask + MySQL + JWT)

A simple **Role-Based Access Control (RBAC)** backend built with **Flask**, **MySQL**, and **JWT authentication**.

This project demonstrates how to:
- Manage **users**, **roles**, and **permissions**
- Protect API routes using **JWT tokens**
- Enforce access using **role-permission mapping**
